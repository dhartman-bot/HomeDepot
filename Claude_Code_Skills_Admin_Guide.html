What Are Skills & Sub-Agents?

The Simple Explanation

Skills are AI workflows that teach Claude your team's exact standards and patterns.

Sub-Agents are specialized AI assistants that handle complex, multi-step tasks autonomously.

Together, they transform Claude Code from a helpful assistant into an expert teammate that knows Home Depot's way of building software.

Skills: Your Team's Knowledge, Codified

Think of skills as expert playbooks. Instead of developers memorizing 50-page standards documents, skills guide Claude to apply those standards automatically.

Example: Payment API Security Skill

Instead of a developer reading PCI compliance docs, the skill tells Claude:

  • "Always encrypt payment data with AES-256"
  • "Validate card numbers using Luhn algorithm"
  • "Log to Splunk with PCI audit tags"
  • "Never log full card numbers - mask to last 4 digits"

Result: Every payment API Claude creates is PCI compliant from line one.

Sub-Agents: Autonomous Task Specialists

Think of sub-agents as specialist teammates. When you need deep exploration, complex searches, or multi-step autonomous work, Claude dispatches a specialized sub-agent.

Example: Explore Sub-Agent

You ask: "How does payment processing work in our codebase?"

Claude launches an Explore sub-agent that:

  • Searches for payment-related files across the entire codebase
  • Reads and analyzes payment service implementations
  • Traces API endpoints to database queries
  • Identifies integration points with payment gateways
  • Returns comprehensive report in one consolidated response

Result: What would take hours of manual exploration happens in minutes.

Common Questions

Q: Do I need to learn how skills work?

A: No! Just use Claude Code normally. Skills activate automatically based on what you're doing. You can ignore that they exist and still benefit.

Q: How do I know if a skill activated?

A: Claude will sometimes mention it ("Applying our payment security standards...") or you can ask: "Did you use a skill for this?"

Q: What if I don't want a skill to apply?

A: Just tell Claude: "Don't apply security checks for this prototype" or "Skip the standards for now." Skills are helpful, not mandatory.

Q: How do sub-agents differ from regular Claude?

A: Sub-agents run autonomously for complex tasks. Regular Claude is conversational. You don't need to choose - Claude automatically uses sub-agents when needed.

Q: Can I create my own skills?

A: Yes! Skills are just markdown files. Copy an existing skill as a template and customize for your team's needs. Ask your team lead.
