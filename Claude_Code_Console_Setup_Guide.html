Console Setup Guide for Pilot Participants
⏱️ Setup Time: 10-15 minutes

Making Your Kickoff Successful

To ensure a productive pilot kickoff session with 50+ developers, complete these preparation steps:

Before the Kickoff

1. Everyone Installs Claude Code (Complete Steps 1-5 Below)

  • Have all participants complete the installation and setup before the kickoff meeting
  • This ensures the kickoff focuses on usage, not troubleshooting installations
  • Target: 100% of participants have Claude Code working by kickoff time

2. Identify Your Champions

  • For a group of 50+ developers, identify 5-10 champions already familiar with Claude Code
  • Aim for 1 champion per 5-10 participants for effective support
  • Champions will provide peer support during and after the kickoff
  • Champions can help troubleshoot issues while the session continues
  • Consider champions who are enthusiastic about AI-assisted development and strong communicators

3. Select Key Use Cases for Live Demos

Choose 2-3 real use cases from your actual work to demonstrate during kickoff:

  • "Debug a production issue" - Show how to analyze stack traces and find root causes
  • "Onboard to unfamiliar code" - Navigate and understand a complex module
  • "Generate unit tests" - Create comprehensive test coverage for existing code
  • "Refactor legacy code" - Modernize old code while maintaining behavior
  • "Review code before PR" - Get AI feedback on quality and potential issues

During the Kickoff

Suggested Agenda (60-minute session):

  • 5 min: Quick installation verification (everyone confirm Claude Code works)
  • 10 min: Review pilot goals and success metrics
  • 30 min: Live demos of 2-3 key use cases with your actual codebase
  • 10 min: Q&A and troubleshooting
  • 5 min: Week 1 action items and champion introductions
💡 Pro Tip: Record the kickoff session so participants can reference the demos later.

1 Access the Anthropic Console

Create Your Account

  1. Navigate to https://console.anthropic.com
  2. Click "Sign Up" or "Log In" if you have an existing account
  3. Complete authentication using:
    • Email/password
    • Google SSO
    • GitHub SSO

Verify Your Account

  • Check your email for verification link (if using email/password)
  • Click the verification link to activate your account
  • Log in to the console

2 Generate Your API Key

Navigate to API Keys

  1. Once logged in, click on your profile/settings in the top right
  2. Select "API Keys" from the navigation menu
  3. Click "Create Key" or "New Key"

Create and Secure Your Key

  1. Name your key (e.g., "Claude Code - [Your Name]")
  2. Click "Create Key"
  3. Copy the key immediately and store it securely
⚠️ CRITICAL: This key will only be shown once. Store it in a password manager or secure note. Never commit it to version control or share it via email/Slack.
Example Key Format:
sk-ant-api03-xxxxx...

3 Install Claude Code

macOS / Linux Installation

curl -sSL https://api.claude.com/install.sh | sh
Alternative (using Homebrew on macOS):
brew install claude

Verify Installation

claude --version

4 Configure Claude Code

Set Up Your API Key

claude config

You'll be prompted to:

  1. Enter your API key (paste the key from Step 2)
  2. Select your preferred model (choose Claude Sonnet 4.5 - recommended)
Model Selection: Claude Sonnet 4.5 offers the best balance of performance and capability for coding tasks, with 1M token context window.

5 Verify Your Setup

Test Claude Code

claude "Create a simple hello world Python script"
Expected Behavior: Claude Code should respond with Python code, no authentication errors, and file operations should work correctly.

Quick Health Check

claude health

6 Explore Basic Features

Try These Commands

1. Interactive Mode
claude
2. Code Analysis
cd /path/to/your/project
claude "Analyze the structure of this codebase"
3. Code Generation
claude "Create a REST API endpoint for user authentication"
4. Code Modification
claude "Add error handling to the database connection code"

Common Issues & Solutions

Authentication Failed

claude config

Re-enter your API key. Verify you copied the full key including the sk-ant-api03- prefix.

Rate Limit Exceeded

Contact your pilot program coordinator to increase limits. Check your usage at: console.anthropic.com/settings/usage

Claude Code Command Not Found

# Restart your terminal, then verify installation
which claude

# If not found, reinstall:
curl -sSL https://api.claude.com/install.sh | sh

Understanding Usage & Billing

Check Your Pilot Credits

As a pilot program participant, you've been provided with credits to get started:

  1. Visit https://console.anthropic.com/settings/billing
  2. Look for the "Credits" section at the top of the page
  3. You'll see:
    • Available Credits: Your current credit balance (e.g., "$100.00")
    • Credits Used: Amount consumed so far
    • Expiration Date: When your pilot credits expire

Where to find this:
Console → Settings (top right) → Billing tab → Credits section

Monitor Your Usage

  1. Visit https://console.anthropic.com/settings/usage
  2. View your current billing period usage
  3. Track API calls, tokens, and costs in real-time
  4. See detailed breakdown by model and date
